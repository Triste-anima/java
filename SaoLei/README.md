扫雷小游戏
=========

1小时用Java开发扫雷游戏:https://www.bilibili.com/video/av71174896
----------------------------------------------------------------

#### 扩展任务：

##### 显示雷的个数

##### 鼠标右键标记雷，标记后无法点击，再次右键取消标记

##### 没有win或者lose的情况下点击banner可以调整难度
>level 0 = 30颗雷

>level 1 = 50颗雷

>level 2 = 80颗雷

>level 3 = 100颗雷
